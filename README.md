# conference-rank
A little project to get feedback from conference attendees

You will need to add a `.env` file:
```
# .env file
SECRET_KEY=your-secret-key
DEBUG=False
DJANGO_ALLOWED_HOSTS=localhost 127.0.0.1 [::1] yourdomain.com
DB_NAME=yourdbname
DB_USER=yourdbuser
DB_PASSWORD=yourdbpassword
DB_HOST=db
DB_PORT=5432
```
